UseCase : UC4.2: Aggiunta nuovo investimento (parte di UC4: Gestione modulo SAVE)
Summary : Permette di aggiungere un nuovo investimento tra quelli disponibili per l’utenza
Actor : Progettista
Precondition : UC4
Postcondition : L’utente visualizza un messaggio di conferma
Base sequence : L’utente naviga nella seconda sezione dedicata agli investimenti, su una barra orizzontale dedicata : L’utente clicca sul tasto dedicato all’aggiunta di un singolo investimento : L’utente compila il form con i parametri richiesti : L’utente visualizza un messaggio di conferma
Branch sequence : In caso di annullamento l’utente visualizza un messaggio di avvenuto annullamento
Exception sequence : no
Note : no
